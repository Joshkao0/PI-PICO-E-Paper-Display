This repository uses the Pi Pico E-Paper Display 2.13" V4B to display a name and a text-based smiley on the screen. The artist's name is shown in red in the lower-left corner of the display.

Features:

Displays a custom name
Shows a text-based smiley
Artist name highlighted in red
Designed for the Waveshare 2.13" V4B e-paper display
